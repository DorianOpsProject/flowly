# flowly

Application de productivité tout-en-un pour les professionnels.

## Stack

- React + Vite
- Tailwind CSS

## Fonctionnalités

- **Landing page** — Hero, features, pricing
- **Tableau de bord** — Vue d'ensemble avec accès rapide
- **Tâches (Kanban)** — Gestion visuelle avec drag & drop, priorités
- **Timer Focus** — Sessions Pomodoro avec historique
- **Notes** — Création et organisation de notes colorées
- **Planning** — Calendrier hebdomadaire avec événements

## Lancer le projet

```bash
npm install
npm run dev
```
